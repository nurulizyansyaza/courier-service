About

A multi-framework delivery cost and time estimation system demonstrating clean architecture across 5 repositories. Each frontend framework (React, Vue, Svelte) shares the same core calculation engine and API backend.

Architecture

Monorepo with 5 packages:

courier-service-core — Shared calculation engine courier-service-api — Express REST API courier-service-cli — Terminal UI (Ink) courier-service-frontend — React + Vue + Svelte courier-service — Orchestration + CI/CD

Tech Stack
