📟 Courier Service CLI

⚠️ STAGING ENVIRONMENT — This is a pre-production deployment for testing.

Interactive terminal UI for the Courier Service delivery cost and time calculator. Built with Ink (React for the terminal).

Setup

Prerequisites

  • Node.js 18 or 20
  • npm
  • courier-service-core must be built first

Quick Start

cd courier-service-core && npm install && npm run build && cd ..
cd courier-service-cli && npm install
npm start

Usage

Commands

CommandDescription
/change mode cost | timeSwitch calculation mode
clearClear screen
helpShow commands
exitExit
